# nodejs与浏览器javascript区别


ECMAScript:  定义了语法，nodejs与浏览器javascript都必须遵守

* 变量定义 循环 函数 闭包...

* javascript使用了ECMAScript语法规范，外加web api 

* nodejs 使用了ECMAScript语法规范，外加nodejs api
